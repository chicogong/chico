# Chico Gong

Personal website built with Next.js 15.

## Links

- Website: [chico.vercel.app](https://chico.vercel.app)
- GitHub: [@chicogong](https://github.com/chicogong)
- X: [@chicogongx](https://x.com/chicogongx)

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS 3

## Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)

## Deploy

Optimized for Vercel:

```bash
npm run build
```
