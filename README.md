# Chico's Portfolio

现代简约的个人主页，采用 Next.js 15 + Tailwind CSS 构建。

## 快速开始

```bash
npm install
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000)

## 部署到 Vercel

```bash
npm install -g vercel
vercel
```

或通过 GitHub 连接到 Vercel 自动部署。

## 技术栈

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion

## 项目结构

```
├── app/           # 页面和布局
├── components/    # React 组件
├── lib/           # 数据配置
└── public/        # 静态资源
```

## 自定义

修改 `lib/data.ts` 更新项目和链接信息。
