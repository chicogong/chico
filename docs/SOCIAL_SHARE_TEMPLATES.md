## 💻 GitHub 优化

### 1. 仓库设置添加网站链接

**步骤**：
1. 访问 https://github.com/chicogong/chico
2. 点击仓库顶部右侧的 ⚙️ "Settings"
3. 滚动到"Website"字段
4. 输入：`https://chico-lac.vercel.app`
5. 勾选"Use your GitHub Pages website"
6. 点击"Save changes"

### 2. 个人资料添加网站

**步骤**：
1. 访问 https://github.com/chicogong
2. 点击右上角头像 → "Settings"
3. 在"About" section找到"Websites"
4. 添加：`https://chico-lac.vercel.app`
5. 点击"Update profile"
