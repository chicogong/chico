# GitHub 设置指南 - 添加网站链接

## 🎯 目标
在GitHub仓库和个人资料中添加网站链接，这样Google和其他AI能更快发现您的网站。

## 📝 步骤1：仓库设置添加网站

### 操作步骤：
1. 访问：https://github.com/chicogong/chico
2. 点击仓库顶部右侧的 ⚙️ **Settings** 图标
3. 向下滚动到 **"Website"** 字段
4. 输入：`https://chico-lac.vercel.app`
5. （可选）勾选 **"Use your GitHub Pages website"**
6. 点击页面底部的 **"Save changes"**

**预计时间**：2分钟

**预期效果**：
- 仓库About section会显示网站链接
- 访问仓库的人能��接点击访问

## 👤 步骤2：个人资料添加网站

### 操作步骤：
1. 访问：https://github.com/chicogong
2. 点击右上角头像 → **"Settings"**
3. 在左侧菜单找到 **"Public profile"**
4. 向下滚动到 **"About"** section
5. 在 **"Websites"** 字段点击 **"Add a website"**
6. 输入：
   - Title: `Portfolio`
   - URL: `https://chico-lac.vercel.app`
7. 点击页面顶部的 **"Update profile"**

**预计时间**：3分钟

**预期效果**：
- 个人资料顶部会显示网站链接
- 提高个人品牌的曝光度

## ✅ 验证步骤

完成上述步骤后，检查：

1. **仓库页面**
   - 访问 https://github.com/chicogong/chico
   - 右侧About section应该显示网站链接

2. **个人资料**
   - 访问 https://github.com/chicogong
   - 顶部应该显示Portfolio网站链接
